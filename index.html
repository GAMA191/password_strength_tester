Easy Integration Guide

This password strength tester can be easily integrated into any website or web application. Follow the steps below:

1. Add the HTML structure

Copy and paste this HTML where you want the tester to appear:

<div id="passwordTesterContainer">
  <div class="password-tester-standalone">
    <label for="passwordInput">Password:</label>
    <input type="password" id="passwordInput" class="password-input" placeholder="Enter password">
    <div class="strength-meter">
      <div class="strength-fill" id="strengthFill"></div>
    </div>
    <div class="strength-text" id="strengthText"></div>
  </div>
</div>

2. Add the CSS

Include this CSS in your stylesheet:

.password-tester-standalone {
  width: 100%;
  max-width: 400px;
  background-color: white;
  border-radius: 10px;
  padding: 1.5rem;
  box-shadow: 0 5px 15px rgba(0,0,0,0.1);
}

.strength-meter {
  height: 8px;
  background-color: #f0f0f0;
  border-radius: 4px;
  margin: 1rem 0;
  overflow: hidden;
}

.strength-fill {
  height: 100%;
  width: 0%;
  border-radius: 4px;
  transition: width 0.3s, background-color 0.3s;
}

.strength-text {
  font-weight: 600;
  font-size: 1rem;
  min-height: 1.5rem;
}
